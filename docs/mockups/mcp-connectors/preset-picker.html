MCP connectors — pick a service from a catalog, paste one token, done

"+ ADD CONNECTOR" opens a two-step settings modal: step 1 is a searchable 2-column grid of preset tiles (hosted vs local, Custom as the dashed escape hatch); step 2 pre-fills everything and asks only for the token. ADD CONNECTOR verifies the connection (lists tools) before closing; the new row flashes in at the top of the list. CONFIGURE on a row reopens the same modal in edit mode (token "Unchanged — paste to replace", REMOVE / SAVE footer).
